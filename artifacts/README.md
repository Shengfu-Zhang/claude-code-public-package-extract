# Artifacts

This directory keeps non-source repository artifacts separated from the browsable `src/` tree.

## Layout

- `original/` contains original published artifacts kept for provenance and reproducibility.
- `extracted/` contains derived artifacts generated from the public package for easier local use.

## Current Files

- `original/claude-code-2.1.88.tgz` — original npm package archive
- `extracted/claude-code-2.1.88-src.zip` — extracted source tree as a convenient download
